# GCR Token Contracts
Smart contracts GCR Token, minted on a vesting schedule.

## Run
- Install: `make install`
- Compile: `make compile`
- Deploy: `make deploy/rinkeby $PRIVATE_KEY` (same script for mainnet)

## Credits
